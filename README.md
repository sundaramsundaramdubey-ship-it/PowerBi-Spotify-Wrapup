# 🎧 Spotify Listening Analysis Dashboard

An interactive **Power BI dashboard** that analyzes Spotify listening data to uncover insights about music consumption, top artists, and listening trends over time.

---

## 📌 Project Overview

This project focuses on analyzing Spotify listening behavior using **Power BI**.
The dashboard transforms raw listening data into meaningful insights through interactive visualizations and KPI metrics.

---

## 📊 Dashboard Features

✔ KPI Cards showing:

* Total Tracks
* Unique Artists
* Unique Albums
* Total Listening Hours
* Total Listening Minutes

✔ **Top Artists Analysis**
Displays artists with the highest listening time.

✔ **Top Tracks Analysis**
Shows the most played songs based on listening minutes.

✔ **Listening Trend Over Time**
Line chart to analyze how listening activity changes over time.

✔ **Interactive Filters**

* Year
* Month
* Artist Name

These filters allow dynamic exploration of the dataset.

---

## 📈 Key Insights

* Identified the **most listened artists** based on total listening minutes.
* Found the **top tracks with highest engagement**.
* Analyzed **listening trends across different time periods**.
